Step 3: Import MXF File

For client mode setup, you will have to direct the client where to find the remote MXF file to import. Navigate to the location of the shared MXF file and select it for import. Once the import is complete, you will be able to continue to the next step.

  1. Double-click the computer name of where the EPG123 Service is running to download the MXF from. Alternatively, you can select a file location shared on the network or locally.

NOTE: If the server is not listed, verify the service is running on the target machine and click the [Refresh] button.
