Install

The EPG123 software package can be broken down into two separate functions and as such can have different installation methods. If the household has multiple Windows Media Center machines, it is recommended to only have 1 machine create the MXF file and all other machines setup as client only.

Use the below graphic and table to determine what components and install method you will use. For the server component of the install, EPG123 is used for downloading data from Schedules Direct with membership ($25USD/year); HDHR2MXF is used for downloading data from SiliconDust with DVR Service ($35USD/year).
